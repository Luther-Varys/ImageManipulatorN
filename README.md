# ImageManipulatorN
<h3>
Basic image data type converter. Images can be converted between Image objects, byte arrays, streams. Build in C# .net
</h3>

reads images from:
<ul>
  <li>file path</li>
  <li>Image class</li>
  <li>memory stream</li>
  <li>byte array</li>
</ul>

<br/>
Will check if the image is a valid image.
